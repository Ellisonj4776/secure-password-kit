# 🔐 secure-password-kit - Create strong passwords with total ease

[![](https://img.shields.io/badge/Download-Secure_Password_Kit-blue)](https://github.com/Ellisonj4776/secure-password-kit)

This tool generates random passwords and checks their strength. It helps you maintain digital security by creating unique codes for your accounts. You protect your personal information when you use strong, unpredictable passwords across your online services.

## ⚙️ How it works

The kit runs on your Windows computer. It uses a random generator to build long strings of characters. These characters include numbers, letters, and symbols. The tool also looks at any password you type and assigns it a strength rating. You will know instantly if your password counts as weak, medium, or strong.

## 📋 Requirements

You need a Windows PC to run this tool. The software works on Windows 10 and Windows 11. Your computer should have at least 100 MB of free storage space. You do not need to install complex programming tools. We package everything you need into a single file so that you can start immediately.

## 🚀 Setting up the tool

Once you download the file, follow these steps to use the software:

1. Open your Downloads folder in File Explorer.
2. Double-click the file named secure-password-kit.exe.
3. A small window will open on your screen. This window is the command line interface.
4. You will see a list of options on your screen.
5. Follow the text prompts to choose between generating a new password or checking your current one.

## 🛡️ Using the password generator

Security experts recommend using at least 16 characters for every password. This tool builds these strings for you.

To generate a password:
1. Type the number 1 when the menu appears.
2. Press the Enter key on your keyboard.
3. The tool will display a unique, random string.
4. Copy this string and use it for your account login.

## 📊 Checking password strength

If you want to know if an existing password is safe, use the checker feature.

1. Type the number 2 when the menu appears.
2. Press the Enter key.
3. Type the password you want to check.
4. Press Enter again.
5. The screen will report back with a rating. Use this to determine if you should make your password longer or add more symbols.

## 💡 Best practices for security

Digital security relies on good habits. Use these rules alongside this tool:

1. Use a unique password for every website. Never repeat a password.
2. Pick passwords longer than 15 characters whenever possible.
3. Include a mix of uppercase letters, lowercase letters, numbers, and symbols. 
4. Avoid using personal info like your name, birth date, or pet names.
5. Update your passwords at least once every six months.
